1. Обучение модели
1) Убедитесь, что файлы .py находятся в том же каталоге, используйте файл "train.py"
2) В "class Config" измените путь к своему набору данных и параметры обучения
3) Запустите функцию main(), чтобы начать обучение
4) После завершения обучения веса модели, словарь и параметры конфигурации по умолчанию сохраняются в "./ ImageCaptioningModel.pth"

2. Вывод (инференс)
1) Убедитесь, что файлы .py находятся в том же каталоге, используйте файл "inference.py"
2) Запустите функцию main(), введите путь к изображению в консоли
3) Будет показано исходное изображение и его описание
